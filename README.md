## How to Run Rustlings

1. Install Rust:
    
    ```bash
    curl https://sh.rustup.rs -sSf | sh
    ```
    
2. Install Rustlings:
    
    ```bash
    cargo install rustlings
    ```
    
3. Start exercises:
    
    ```bash
    rustlings watch
    ```
